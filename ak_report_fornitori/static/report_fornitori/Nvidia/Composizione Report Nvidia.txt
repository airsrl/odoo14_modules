Quinta Colonna: Product Description
Sesta Colonna: Quantity On Hand = quantità di prodotto in casa tra i vari magazzini ( sede - negozio )
Settima Colonna: PO Back Log = Quantità di prodotto in Arrivo da fornitore ( al momento non è gestita dall'ufficio acquisti tramite ordine fornitore)

I dati saranno poi cosi strutturati in tabella:
Prima Colonna: Reporter ID = Ak Informatica
Seconda Colonna: Ean del proddoto 
Terza Colonna: Manufacturer
Quarta Colonna: Product Description
Quinta Colonna: Quantity Sold/Returned = qunatità emessa in fattura 
Sesta Colonna: Sold/Invoice Date = DAta della fattura
Settima Colonna: Invoice Number
